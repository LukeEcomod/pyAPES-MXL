Kersti Haahti, Luke 30.1.2018

Meteorological data from FMI Jokioinen station downloaded from FMI open data 
using FetchFMIOpen (https://github.com/AaltoUrbanWater/FetchFMIOpen)

Missing values = -999
Gap-filling by repeating last non-missing value - Should this change?
(see H:\Lettosuo\Forcing_data\FMI_jokioinen.xlsx)

	yyyy,mo,dd,hh,mm - year, month, day, hour, minute
	Prec - Precipitation [mm/dt]
	Tair - Air temperature [degC]
	U - Wind speed 10 min avg. [m/s]
	RH - Relative humidity [%]
	Rg - Global radiation [W/m2]
	Press - Pressure, msl [hPa]
	SnowD - Snow depth [cm]
	LwR - Longwave radiation [W/m2]
